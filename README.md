# Spring_Web_Project

프레임워크 : springframework

자바 : 1.8ver

DB : Mariadb / mybatis

UI : Bootstrap / kendo ui

Was :  Tomcat v9.0


- spring framework 를 통해 신발 쇼핑몰을 제작중입니다
- 현재 적용기능
- 상품 명 검색(검색후 나오는 페이징Ajax) / 상품 페이징 처리 / 해당 상품 댓글 처리(Ajax) 익명/ 상품 댓글 페이징 처리 및 댓글 CRUD(Ajax)
- 사이즈 클릭시 갯수 및 주문칸 자동생성 및 jquery처리 / 상품 데이터 crud
- 상품 상세 설명 이미지 파일 업로드 가능 / 상품 관련 정보 워드/ppt등 이미지가 아닌 파일 별도로 상단부 첨부파일로 지정 및 다운 / 보안에 취약한 압축파일 업로드불가

- 진행중인 기능(첨부파일 삭제/수정//Quartz) 
- 차후 진행할 기능(Spring Web Security통한 로그인 및 권한부여/장바구니 /AWS EC2 서버배포)
