# Spring_Web_Project

프레임워크 : springframework

자바 : 1.8ver

DB : Mariadb / mybatis

UI : Bootstrap / kendo ui

Was :  Tomcat v9.0


- spring framework 를 통해 신발 쇼핑몰을 제작중입니다
- 현재 적용기능
- 상품 명 검색(검색후 나오는 페이징Ajax) / 상품 페이징 처리 / 해당 상품 댓글 처리(Ajax) / 상품 댓글 페이징 처리 및 댓글 CRUD(Ajax)
- 사이즈 클릭시 갯수 및 주문칸 자동생성 및 jquery처리 / 상품 데이터 crud(이미지 업로드가 필수 반만가능)

- 진행중인 기능(상품 상세 이미지 업로드 (uuid/ajax/thumbnail)/첨부파일 다운로드/Quartz) 
- 차후 진행할 기능(Spring Web Security통한 로그인 및 권한부여/장바구니 /AWS EC2 서버배포)
